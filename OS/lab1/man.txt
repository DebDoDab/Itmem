Приколы для деда
~~~~~~~~~~~~~~~~~
Функции приколов для деда
    calc - кулькулятор
        Аргументы: operation num1 num2
        operation = "sum"/"div"/"sub"/"mul"
        num1 и num2 - инты

    search - Поиск
        Аргументы: dir regexp
        dir - папка
        regexp - гулярное выражение

    reverse - esrever
        Аргументы: inpfile outfile
        inpfile - файл для чтения, обязан существовать
        outfile - файл для вывода

    strlen - 6(4)
        Аргументы: str
        str - а, длину которой выведет это прекрасное приложение

    log - Логи
        Аргументы: няма
        Читает логи из файла /var/log/anaconda/X.log. При этом сначала выводятся warning, затем information

    exit - дверь
        Необязательные аргументы: code
        code - код ошибки двери, если кода нет - выходим в 0

    interactive - интерактивчик
        Аргументы: нет
        Да.


Коды ошибок:
    0 - все ок
    1 - не ок, неправильное количество аргументов
    2 - не ок, аргумент неправильный
    3 - не ок, файла/папки для чтения не существует
    4 - дай пермишны
    5 - что-то пошло не так с чем-то
    6 - дай фулл
